# Matplotlib Basics – Data Visualization in Python

This repository demonstrates the fundamentals of **Matplotlib**, a popular Python library used for data visualization. It includes examples of different types of plots such as line graphs, scatter plots, bar charts, and pie charts.

---

## Overview

Matplotlib is a powerful plotting library used to create static, animated, and interactive visualizations in Python.

This project covers:
- Importing Matplotlib
- Using `pyplot`
- Line graphs
- Scatter plots
- Bar charts
- Pie charts
- Graph customization (titles, labels, legends)
